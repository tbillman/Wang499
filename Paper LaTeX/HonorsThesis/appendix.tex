\subsection*{A. Sample Program Inclusion}
\addcontentsline{toc}{subsection}{A. Sample Program Inclusion}

This is a procedure to compute the sum using Maple.
\begin{verbatim}
  SUM := proc(n)
      local i, total;
      total := 0;
      for i from 1 to n do
         total := total+i;
      end do;
      total;
  end proc;
\end{verbatim}
